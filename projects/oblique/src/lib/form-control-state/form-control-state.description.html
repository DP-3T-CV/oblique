Adds control-mandatory class to highlight required elements. Also controls the has-error class to style the component in case of errors

Required elements will be discovery by either required attribute on the related form control, mandatory input on the directive or ngControl.errors.required error on the related form control.

Works well with the or-error-messages component which displays the errors.

Deprecated with Material Design

If Oblique's theme hasn't been changed to Bootstrap, the Angular implementation should be preferred.

As a deprecated component, it is not likely to receive any new features or improvements.

This component won't be removed as long as Bootstrap is still supported.
