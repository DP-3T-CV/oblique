The themes service allows the live switching between Material and Bootstrap themes. It even accepts custom themes. The given theme will be automatically loaded and applied.

This feature should only be used when live theme change is required. Otherwise, simply load the desired theme in angular.json

Per default, Oblique tries to load the Frutiger font. But because of licence restrictions, those files cannot be delivered with Oblique. Each project has to obtain the font files itself from webforum@bk.admin.ch or the federal chancellery intranet. Each project is also responsible for the font protection according to its licence. The proposed solution consist of only delivering the font if the Referer Http header is whitelisted.
