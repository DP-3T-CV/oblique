The Multiselect component allows you to select multiple elements, using checkboxes in a dropdown menu.

Deprecated

This component is meant to be used with Bootstrap. Material theme should instead use the Angular implementation demonstrated below.

As a partially deprecated component, it won't be prioritized for new features or improvements but won't be removed as long as Bootstrap is supported.
